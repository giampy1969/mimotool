This section is dedicated to the presentation and the analysis of the characteristics of the current model: representation forms, structure in the frequency domain, step and impulse responses, frequency behavior, etc [3], [4]. While the most important global properties are resumed in the main window (dimensions, controllability, observability, stability), it's possible to analyze in detail all the properties and characteristics of the model by means of the view menu and of the analysis menu: every command of these menus opens a new subsection made up of one or more different windows in which the user can move using the general buttons BACK, NEXT, CLOSE or other specific buttons depending on the selected functionality.
